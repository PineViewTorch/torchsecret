TorchSecret Image Search

For use by staff members to find images for postcards

Searches Google Images for items "labeled for reuse"

Important! Make sure that, on finding an image, you do all of the following:

  1. Read the license - make sure it does, in fact, allow you to use the image. (On Flickr, for example, find the link on the right sidebar that says "some rights reserved".) Note the following:
    • You may use images marked for non-commerical purposes (slash through dollar sign).
    • You may not use images that disallow derivative works (equals sign).
  2. Write down the author of the image - if the license requires attribution, which it likely will, the author will have to be listed somewhere in the paper.

This information must be included on submitting your postcard!

If you just grab an image, lose the page, and then submit your secret, you will be forced to start from scratch, as we can not use any postcard that violates copyright, and attribution is a key part of most re-use licenses.

Questions? Ask them.
